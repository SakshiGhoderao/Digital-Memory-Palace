# Digital-Memory-Palace
A beautiful and interactive Digital Memory Palace web app built using HTML, CSS, and JavaScript that allows users to store memories inside rooms with emotions, images, and themes.  It works like a personal memory journal with a modern UI and dark mode.
🌐 Live Features
✨ Create unlimited rooms
✨ Save memories inside rooms
✨ Add emotions to memories
✨ Upload images
✨ Filter memories by emotion
✨ Dark Mode support 🌙
✨ Memory insights dashboard 📊
✨ Data saved in browser using LocalStorage
✨ Smooth animations and transitions

Technologies Used
 HTML5
 CSS3
 JavaScript (Vanilla JS)
 LocalStorage
